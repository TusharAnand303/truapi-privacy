Privacy Policy for TruAPI - API Tester

Last Updated: November 10, 2025

Overview

TruAPI is a client-side API testing tool. We do not collect, transmit, or have access to any user data.

Data Storage

  • All data (API URLs, headers, request bodies, authentication tokens, responses, history, and presets) is stored locally on your device only using Chrome's storage API
  • No data is ever transmitted to our servers or any third parties
  • You have complete control over your data and can clear it at any time through the extension interface

Data You Input

While using TruAPI, you may input sensitive information such as:

  • API URLs and endpoints
  • Authentication tokens and API keys
  • Request headers and bodies
  • API responses

Important: All this data remains on your device only. We (the developers) never receive, access, or transmit this information.

Third-Party Services

TruAPI makes HTTP requests directly to API endpoints that you specify. These requests are made from your browser to those endpoints. We do not intercept, monitor, or store these communications.

Data Collection

TruAPI does not collect:

  • Personal information
  • Usage analytics
  • Crash reports
  • Any form of telemetry

Changes to This Policy

We may update this privacy policy. Any changes will be posted with an updated revision date.
